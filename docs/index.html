同一道题，各写各的，择优采纳

      
一条提示词并行发给多个智能体。开启 worktree 隔离后各自在独立副本里改代码，改动汇成文件 × 改动块的看板——按 hunk 勾选合并，不必整版全收。

      

        

          
CLAUDE 的版本

          
+ retry(3, backoff=exp)
− retry(1)
✓ 采纳此块

        

        

          
CODEX 的版本

          
+ timeout=30s, jitter
− timeout=5s
✓ 采纳此块

        

      

    

  


  


  
  

    

      ⠿
      
      裁决
      证据链
      − ⤢ ×
    

    

      
争不出结果？交给裁判，留下证据

      
把几个终端的画面冻结成对照卡片，交给另一个 CLI 或 daemon 裁判：结论 / 理由 / 把握程度结构化返回。封存后连同当时的 git 基线存成私有检查点，之后可检查代码漂移、对比派生分支——每一步显式操作，Relay 不替你按回车、不改写记录、不自动评价。

      

        冻结现场只读快照→
        裁决人握回车→
        封存显式点击→
        检查点git 基线→
        血缘 / 漂移多代对比
      

    

  


  


  
  

    

      ⠿
      
      审批
      USER GATE
      − ⤢ ×
    

    

      
闸门和回车，都在你手里

      
所有任务的工具审批汇入同一扇审批窗，处理完各流程自动继续；跨终端填入一律 bracketed paste——只填入、不执行。Relay 自身没有账号、没有遥测、没有云端。

      

        
终端 PTY从不记录内容，只观察是否在输出

        
画面快照仅存进程内存，不进剪贴板、不写盘

        
决策检查点显式保存才落盘：目录 0700 · 文件 0600 · 删除进废纸篓

        
网络Relay 自身零远端——LOCAL ONLY

        接入
        零代码 manifest
        − ⤢ ×
      

      

        
// 按行输出走 generic，讲 ACP 走 acp
{
  "id": "echo",
  "requirements": [{
    "env": "ECHO_BIN"
  }],
  "generic": {
    "command": "ECHO_BIN",
    "resume_arguments":
      ["--resume", "{session}"]
  }
}

        
一份 JSON 接入任何 CLI；声明会话参数即获得续聊。
